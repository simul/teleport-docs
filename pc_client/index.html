2.1. Windows PC Client¶

The Reference Client for Microsoft Windows can operate in desktop or VR mode.

2.1.1. File Storage¶

Temporary files are stored in the current user’s local AppData folder. Files that are permanent are installed with the application to assetslocalGeometryCachetextures under the installation folder.
